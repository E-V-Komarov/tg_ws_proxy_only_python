# tg_ws_proxy_only_python

## Установка
### **Iphone**
1. Скачайте из маркета [приложение `iSH`](https://apps.apple.com/ru/app/ish-shell/id1436902243)

> [!IMPORTANT]
> **Важно!** Установка пакетов в iSH не работает без vpn ;(

2. Впишите в терминал команду и выполните:
```bash
apk update &&
apk add python3 py3-pip py3-cryptography curl &&
printf "cat /dev/location > /dev/null &\npython3 tg_ws_proxy_cli.py" >> ~/.profile &&
curl -L -o tg_ws_proxy_cli.py "https://github.com/E-V-Komarov/tg_ws_proxy_only_python/releases/download/0.0.1/tg_ws_proxy_cli.py"

```
Команда установит вам python, базовые приложения, а также установит автоматический скрипт для запуска локального прокси. Для запуска локального прокси нужно просто закрыть приложение `iSH` и заново его открыть.

Прикол №1: требуется оставить для приложения "Местоположение - всегда, не точное" (Чтобы приложение работало в фоне, _НИКАК_ не используется в самом приложении)

Прикол №2: можно сделать автоматизацию при помощи `Команды` -> `Автоматизация`:  `Когда приложение "Telegram" открыто` -> `Запуск после подтверждения` -> `Сделать "Открыть приложение iSH"`; Это позволит спрашивать открытие приложения, каждый раз когда открываете телегу, чтобы не искать этот `iSH` среди приложений, ну и чтобы не засорять главный экран :)

Прикол №3: требуется добавить [свой локальный прокси](t.me//socks?server=127.0.0.1&port=25566) в список своих прокси для телеграмм.
